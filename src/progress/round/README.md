# Round Progress
This library creates a graphical rounded progress in a easy way, implementing an HTML Canvas.

### Example
```HTML
 <round-progress
     id="1"
     value="50"
     radio=30"
     stroke=""
     data='{"hola": 123}'
     style="color: #FCB13F; font: 25px Chiller, sans-serif"
 ></round-progress>
```
